# Application Root

This directory contains the core application code.

- `api/`: API route definitions and endpoints.
- `core/`: Core application configuration, settings, and security logic.
- `db/`: Database connection, session management, and migrations.
- `models/`: SQLModel database models.
- `schemas/`: Pydantic schemas for request/response validation.
- `services/`: Business logic and service layer.
- `main.py`: The entry point of the FastAPI application.
